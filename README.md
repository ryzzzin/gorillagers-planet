# Gorillagers Planet

Gorillagers Planet is a Fabric mod for Minecraft `1.21.11` that adds a new animated mob called Gorillager and jungle-themed gameplay around it.

## Features

- Adds the Gorillager mob with GeckoLib model and animation support.
- Includes `walk` and `idle` animations for the Gorillager.
- Natural Gorillager spawning in jungle biomes in groups of `1-4`.
- Adds a Gorillager Spawn Egg item with a custom icon.
- Cookie barter behavior: throw a cookie to a Gorillager, it holds it for a moment and throws back a reward.
- Adds `Banana` as a food item (currently used as barter reward).
- Uses custom model, animation, and texture assets for the mob, items, and spawn egg.

## Version

Current mod version: `1.1.0`.
