# Gorillagers Planet

Fabric мод с мобом `Gorillager` (Гориллянин), GeckoLib-анимациями и яйцом спавна.

## Run

```bash
./gradlew runClient
```

Проверка в игре:
- `/summon gorillagers_planet:gorillager`
- `/give @p gorillagers_planet:gorillager_spawn_egg`
